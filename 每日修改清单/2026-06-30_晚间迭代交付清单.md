## 三、协作开发者快速上手

### 3.1 拉取最新代码

```bash
git pull origin main
```

### 3.2 配置本地环境

**重要：`application-dev.yml` 已从 Git 移除，本地需要重新创建**

```bash
# 步骤 1: 复制示例配置
cp 软件实训项目-code/backend/src/main/resources/application-dev.example.yml \
   软件实训项目-code/backend/src/main/resources/application-dev.yml

# 步骤 2: 修改以下配置
# - deepseek.api-key: 填入你的 DeepSeek API Key
# - spring.datasource.password: 填入你的 MySQL 密码
# - spring.mail.username/password: 填入你的邮箱配置（可选）
```

### 3.3 验证功能

**简历优化验证**：
1. 启动后端：`mvn spring-boot:run`
2. 访问前端简历优化页面
3. 上传 PDF/DOCX/TXT 简历
4. 点击「AI 优化」
5. 验证返回结果中包含「AI 优化版本」区块

**职业探索验证**：
1. 新用户（无画像无测评）进入职业探索
2. 验证出现追问提示
3. 完成测评后，从测评结果页进入探索
4. 验证推荐方向基于测评维度
